# CET Typing Lab

CET Typing Lab is a browser-based typing practice site for CET4 / CET6 English learners. It combines English sentence and passage typing with instant speed, accuracy, sound feedback, and shareable results.

## Visit the Website

Open the public site here:

https://L-zeus.github.io/cet-typing-lab/

A physical keyboard is recommended for the best experience.

## What You Can Do

- Practice CET4 / CET6 style English sentences and full reading passages.
- Track WPM, accuracy, progress, elapsed time, weak keys, and slow words.
- Switch punctuation, case sensitivity, blind typing, and stop-on-error options.
- Use mechanical keyboard sound packs while typing.
- Review Chinese translations, core phrases, syntax notes, and CET vocabulary.
- Copy a share card after a round and invite friends to test the site.

## Who It Is For

This site is mainly for students preparing for CET4 / CET6 who want to improve English input fluency, reading focus, and typing rhythm in a lightweight web environment.

## Privacy

Practice data and local accounts are stored in your browser. This public repository contains only the built static website files used for GitHub Pages.

The original source code is kept in a private repository. This public repository is only for hosting the compiled website.
